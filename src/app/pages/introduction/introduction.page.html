Introduction

ionic-angular-theme-switch

What does the library do?

ionic-angular-theme-switch service can be used to easily change your app's theme on the fly.
The actual theme will be stored and restored on reload or revisitation.

To simplify usage, this library provides a component to toggle between two themes.

What does this app do?

This app aims to provide some examples on how to implement the service and the component into your ionic angular app.

When navigating, the theme is reset to default one in this app. Thus each demo page should be seen as a single app.
On reload or revisitation of the last opened demo page, theme is going to restored as it would do in any other app.
 
This inconvenience is required to present all variants of library usage in one app.

How do I run this app?

  1. Clone GitHub repository jonathanheilmann/ionic-angular-theme
    ($git clone https://github.com/jonathanheilmann/ionic-angular-theme.git)
  2. Navigate $cd ionic-angular-theme
  3. Install dependencies $yarn install
  4. Build library $ng build ionic-angular-theme-switch
  5. Run $yarn start
